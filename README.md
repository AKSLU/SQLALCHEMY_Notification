# SQLALCHEMY_Notification

Простой проект на Flask + SQLAlchemy для создания и просмотра уведомлений.

---

## Требования

* Python 3.10+
* Библиотеки:

  * `Flask`
  * `SQLAlchemy`

Установите через pip:

```bash
pip install flask sqlalchemy
```

---

## Структура проекта

* `app.py` — основной скрипт с Flask-приложением.
* `templates/admin.html` — форма добавления уведомления.
* `templates/user.html` — отображение уведомлений пользователю.
* `notifications.db` — база данных SQLite с таблицей `notifications`.

Таблица `notifications` имеет колонки:

* `id` — первичный ключ
* `text` — текст уведомления
* `author` — автор уведомления
* `timestamp` — время создания (автоматически)

---

## Использование

1. Клонируйте репозиторий:

```bash
git clone https://github.com/AKSLU/SQLALCHEMY_Notification.git
cd SQLALCHEMY_Notification
```

2. Запустите Flask:

```bash
python app.py
```

3. Откройте в браузере:

* Админка: [http://127.0.0.1:5001/admin](http://127.0.0.1:5001/admin) — добавление уведомлений.
* Пользовательский интерфейс: [http://127.0.0.1:5001/user](http://127.0.0.1:5001/user) — просмотр уведомлений.

---

## Пример работы

* Добавление уведомления в админке:

```
Текст: Тестовое уведомление
Автор: Admin
```

* Просмотр уведомления у пользователя:

```
Admin — 2025-12-06 01:23:45
Тестовое уведомление
```
